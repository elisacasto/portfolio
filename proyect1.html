The project

Madrid 360º is a personal project, born from the tourist guide I designed in 2016 for my final degree thesis. I really liked working on this project, but I realized that when I finished that tourist magazine, I wanted to continue with this project and make it more practical than a book, to design an experience which would be closer to the user, and above all, an experience for everyone!

The process

First, I started doing a study on other tourism websites. My first objective was to focus on a problem and to try to find a way to fix them by interviewing the potential users. Once I prioritized the content with user stories I started designing my website with the help of the Mobile First methodology. My process' chronology with each single user's story has been: wireframes, tests with potential users, digital design and finally programming. Choosing the style has not been so difficult as I wanted. I wanted to keep the one I chose for the tourism magazine. A very clean style with contrasting colors and linear typography. This project is still in a process state.
