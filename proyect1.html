Madrid 360°

Category: UI/UX - Front end

Client: Personal project

Year: 2019

Visit website

Madrid 360º is a personal project, born from the tourist guide I designed in 2016 for the end of degree work. I really liked working on this project, but I realized that when I finished that tourist magazine, I wanted to continue with this project, and make it more practical than a book, to design an experience closer to the user, and above all, for everyone!

The process

I've started doing a study on other tourism websites. My first objective was to focus on the problem, and try to find the first solutions by interviewing the potential users. Once, I prioritized the content, with user stories, I started designing with the Mobile First methodology. My chronology of the process with each single user story has been: wireframes, tests with potential users, digital design and finally programming. Choosing the style has not been so difficult. I wanted to keep the one I chose for the tourism magazine. A very clean style, contrasting colors and linear typography. This project is still in a process state.
